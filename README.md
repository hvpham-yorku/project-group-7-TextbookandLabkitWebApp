\# Rebook YorkU – Textbook \& Lab Kit Exchange Platform



Rebook YorkU is a web application built using Java, Maven, and Spring Boot.  

The goal of the platform is to allow York University students to buy, sell, and manage textbooks and lab kits within the university community.



---



\## Project Overview



This application includes:



\- User Sign-In (YorkU authentication style)

\- Login and Logout functionality

\- User Profile page

\- Basic listing management features

\- Backend service layer using Spring Boot

\- Thymeleaf templates for frontend rendering



---



\## Technologies Used



\- Java

\- Maven

\- Spring Boot

\- Thymeleaf

\- JUnit (for testing)



---



\## How to Run the Project



1\. Open the project in your IDE

2\. Run:



&nbsp;  mvn clean install



3\. Then run:



&nbsp;  mvn spring-boot:run



4\. Open your browser and go to:



&nbsp;  http://localhost:8080/login
